# Color Picker

A simple color wheel picker built with React and Vite.

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Features
- Draws an HSL color wheel on a canvas
- Hover color preview

## Run
```bash
npm install
npm run dev
```

Open the app at `http://localhost:5173`.

## Main File
- `src/App.jsx` — main app drawing the color wheel
